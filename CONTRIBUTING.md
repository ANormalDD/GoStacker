# 贡献指南

感谢你对 GoStacker 感兴趣！我们欢迎任何形式的贡献，包括提交 Bug、改进文档或提交代码。

## 如何贡献

1.  **Fork** 本仓库。
2.  创建你的特性分支 (`git checkout -b feature/AmazingFeature`)。
3.  提交你的修改 (`git commit -m 'Add some AmazingFeature'`)。
4.  推送到分支 (`git push origin feature/AmazingFeature`)。
5.  开启一个 **Pull Request**。

## 本地开发

请参考 `README.md` 中的 [快速开始] 章节启动本地开发环境。

## 代码规范

-   请遵循 Go 官方的代码规范。
-   提交代码前请运行 `go fmt ./...`。
